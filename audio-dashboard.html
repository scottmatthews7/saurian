Saurian — Audio Audit Dashboard

Plays every sound the game currently uses, driven through the real audio engine (createAudio() from src/audio.js) calling the exact same methods the game loop calls — so what you hear is what plays in-game. Use it to flag the ones that sound wrong.

Each row marks whether the sound is a REAL SAMPLE (with its source from CREDITS.md) or PROCEDURAL synth, plus a one-line note on where it's used.

⚠ This page MAKES SOUND on every Play click (it is an audio auditor). The first click also unlocks the browser audio context. Mind your volume on calls.
Click any Play button to unlock audio and start auditing.
Engine: src/audio.js · sample paths + tunables: src/config.js → AUDIO · sample sources: CREDITS.md. Real samples live in assets/audio/. Sounds marked PROCEDURAL are synthesised live in the Web Audio API (no file). Sounds marked BEING REMOVED still fire in the current build but are slated to go.
